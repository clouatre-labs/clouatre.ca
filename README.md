# Hugues Clouatre - Personal Website

Professional website and blog for Hugues Clouatre, showcasing technology leadership, AI workflows, and cloud architecture insights.

## Tech Stack

- **Framework:** [Astro](https://astro.build/) with [Astro Paper](https://github.com/satnaing/astro-paper) theme
- **Styling:** Tailwind CSS
- **Deployment:** GitHub Pages
- **Content:** Markdown/MDX with type-safe frontmatter

## Features

- SEO-friendly and accessible
- Light/dark mode toggle
- Fast performance (static site generation)
- Responsive design
- Blog with content collections
- Static search with Pagefind
- RSS feed
- Automatic OG image generation

## Development

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview
```

## Deployment

The site is automatically deployed to GitHub Pages on push to `main` branch.

**Live URL:** https://clouatre-labs.github.io/clouatre.ca/

## Security

- Gitleaks secret scanning in CI/CD
- No analytics tracking (privacy-first)
- HTTPS only
